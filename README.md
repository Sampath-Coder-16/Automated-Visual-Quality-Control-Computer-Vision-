# Automated Visual Quality Control (Computer Vision)

## Overview
This project implements an AI-based quality inspection system that detects product defects using deep learning and computer vision.

## Features
- Defect classification (Pass / Defect)
- Transfer learning (MobileNetV2)
- Real-time inference using OpenCV
- Grad-CAM visualization for explainability

## Dataset
- NEU Surface Defect Dataset
- PCB Defect Dataset
